# FM26 Mobile Wonderkids

FM2026 Mobile 기준 원더키드/즉전감 선수 정적 HTML 페이지입니다.

## 보기

https://donghyeon-dev.github.io/fm26-mobile-wonderkids/

## 구성

- `index.html`: 단일 정적 페이지
- 상단 1: 포텐순 TOP 12 — 가격/영입 난이도 제외, 성장성 우선
- 상단 2: 즉시 주전감·가성비 TOP 10 — 비싸지 않고 바로 1군/로테이션에 넣기 좋은 선수
- 전체 표: 가성비 우선순위, 선수 검색, 포지션 필터, 가격대 필터, 즉전감 필터 포함

## 데이터 주의

- FMM Vibe FM26 Mobile Wonderkids 리스트와 FM26 Mobile 추천 글을 바탕으로 정리했습니다.
- 실제 가격/존재 여부/능력치는 세이브, 선택 리그, 패치 버전에 따라 달라질 수 있습니다.
- iOS에서는 게임 내 검색용 쇼트리스트로 사용하세요.
